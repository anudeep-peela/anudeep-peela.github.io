Design Optimization & CAD Automation

Engine Parts Allocation & Layout Optimization

Python Ray Genetic Algorithms Catia CAD 3D Geometry Heuristics
Ray Scale Catia CAD MVP

Helped largest automobile client in building their next-gen engines by optimizing engine parts allocation and piping. Developed a generalized automatic end-to-end layout optimization MVP integrated directly into the client’s CAD workstream.

Problem & Aim

The primary aim was to develop an AI/ML-based design layout optimization tool. This tool optimizes the arrangement and piping of various mechanical components in a given space, ensuring that packaging constraints are met and the layout is volume-efficient.

Context

Physical engine design is highly constrained. We aimed to automate the packaging optimization of layouts and pipe routing, converting highly manual design processes into a scalable digital pipeline.

Approach

Classified the layout challenge as an NP-Hard search problem. We limited the solution space by constraining part movements and fixing each part sequentially before adding new ones, performing layouts in heuristics-driven test and learn loops.

Technical Choices

Genetic Algorithm: Smartly determines x,y,z coordinates and rotation along x,y,z for all parts, using cost function logic to reinforce potential solutions.
Heuristics: Incorporated intuitve order arranging rules from expert designers to find optimal layouts faster.
Cost Function: Balances pipe cost and volume, adjusting weights dynamically based on experimental data.
Piping Algorithms: Developed fast and robust routing algorithms to automatically connect pipes while avoiding 3D obstacles.
Ray Parallelism: Code runs on personal laptops and distributed cloud nodes, using Ray for distributed parallel computing.
Pre/Post Processing: Includes object simplification, helper objects, data cleanup, and automated resolution recovery, exporting native .stl files directly into Catia CAD software.

Impact

Improved Design Quality & Speed: Fast development times with agile evaluation of complex packaging layout variations.
Cost & Time Savings: Automated layout experiments with minimal programming knowledge, saving manual engineering hours.
Flexibility: Config files act as a simple communication medium, allowing mechanical designers to adjust parameters without writing Python code.
Seamless Tool Integration: Effortless CAD file import/export via native .stl support, facilitating collaborative engineering workstreams.

What I Learned

Automating mechanical parts layout requires balancing mathematical search algorithms (Genetic Algorithms, NP-Hard solvers) with designer intuition. Config-driven interfaces are key to achieving user-friendliness and trust with domain engineers.
