Design Optimization & CAD Automation

Engine Parts Allocation & Layout Optimization

Python Optimization Algorithms CAD Systems 3D Geometry Heuristics Distributed Compute
CAD Automation Layout Optimization

Helped one of the largest automobile clients in building their next-gen engines by optimizing engine parts allocation and piping. Developed an MVP for generalized automatic end-to-end layout optimization to be integrated into the client's workstream.

Aim & Problem

The primary aim of the project was to develop a design layout optimization tool. This tool is intended to optimize the arrangement and piping of various mechanical components in a given space, ensuring that spatial constraints are met and layouts are highly efficient in terms of volume.

Context

Partnered with one of the largest global automotive manufacturers to optimize mechanical packaging layouts, converting manual design iterations into highly automated pipelines.

Approach

Formulated layout optimization as a constrained spatial search problem. We limited the solution space by iteratively constraining part movements and fixing components sequentially to test and learn rapidly.

Technical Choices

Algorithmic Search: Used optimization algorithms (such as genetic algorithms and heuristic methods) to intelligently determine coordinates and rotations for all parts, guided by cost functions to reinforce potential solutions.
Heuristics: Incorporated intuitive layout rules from expert designers to arrive at feasible arrangements faster.
Piping Algorithms: Developed routing algorithms to automatically connect pipes while avoiding spatial obstacles.
Compute Scalability: Structured the pipeline to run on both personal laptops and distributed cloud compute environments for parallel execution.
CAD Integration: Integrated automated pre- and post-processing tools to clean up data and enable import/export support with standard CAD software (such as Catia).

Impact

Efficiency & Agility: Allowed for faster development times and the evaluation of more design variations in a highly agile manner.
Design Feasibility: Incorporating domain expertise and heuristics from designers ensured that layouts were highly optimized and practical.
User-Friendliness: Configuration files act as an easy interface for parameters, allowing designers to run layouts without programming knowledge.
Tool Integration: Seamless integration with CAD software facilitates collaboration and further design refinement across departments.

What I Learned

Automating complex physical layouts requires combining mathematical search methods with flexible configuration interfaces that respect mechanical designers' real-world workflows and intuition.
